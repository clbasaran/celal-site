Brand Guidelines

Apple Design Language 2025 implementation standartları

✅ Best Practices

  • • SF Pro font stack kullanımı
  • • 8pt grid system ile spacing
  • • Semantic color usage
  • • Consistent border radius (8px, 12px, 16px)
  • • Apple motion principles (0.15s, 0.25s, 0.4s)
  • • Dark/light mode support
  • • Accessibility compliance

❌ Avoid

  • • Custom fonts dışında font kullanımı
  • • 8pt grid dışında spacing
  • • Gradients, shadows, 3D effects
  • • Inconsistent border radius
  • • Çok hızlı veya yavaş transitions
  • • Hard-coded colors
  • • Mobile-first design ihmal

Implementation

CSS custom properties ile modern, maintainable code structure

CSS Usage

/* Import brand design system */
@import 'assets/style/brand-style.css';

/* Usage examples */
.my-component {
  padding: var(--space-4);
  background: var(--color-bg-elevated);
  border-radius: var(--radius-lg);
  color: var(--color-text-primary);
  font-family: var(--font-system);
}

/* Button implementation */
.custom-button {
  @extend .btn, .btn-primary;
  font-size: var(--text-lg);
}
